## ğŸ” 2.2.3 Reading â€“ **Search Algorithms**

### â¤ Sequential (Linear) Search

* Works on **unsorted lists**
* Check each item until you find the target or reach the end
* **Time complexity**: **O(n)**

### â¤ Binary Search

* Works only on **sorted lists**
* Repeatedly divide the list in half

### ğŸ§  Binary Search Algorithm:

```py
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```

* **Base condition**: search fails if `low > high`
* **Time complexity**: **O(log n)** â€” very efficient
